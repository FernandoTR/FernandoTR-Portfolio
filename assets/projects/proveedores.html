Proyecto empresarial SAP Business One Integración ERP

Portal de Proveedores.

Portal web empresarial para validar CFDI de proveedores, asociarlos con documentos registrados en SAP Business One y centralizar su seguimiento.

Tecnologías:
ASP.NET MVC C# SQL Server SAP Business One SAP DI API JavaScript jQuery Bootstrap HTML/CSS
Problema

Era necesario validar XML, PDF, UUID, importes y datos fiscales antes de asociar documentos de proveedores a registros existentes en SAP Business One.

Solución

Se implementó un portal web que automatiza la validación de CFDI, evita duplicidad documental y permite a proveedores consultar el estatus de pago de sus facturas.

Características principales

  • Carga y validación de XML y PDF de proveedores.

  • Asociación automática con órdenes de compra y facturas en SAP B1.

  • Consulta de estatus de pago para proveedores.

  • Repositorio de documentos XML y PDF.

  • Soporte multiempresa (multi-sociedad).

Mi rol

  • Desarrollo backend en ASP.NET MVC.

  • Construcción de interfaces y flujo funcional.

  • Validaciones XML conforme al Anexo 20 del SAT.

  • Integración con SAP Business One.

  • Consultas SQL y reglas de negocio.

  • Flujo de carga y consulta de facturas.

Retos técnicos

  • UUID único por documento.

  • Validación XML conforme al SAT.

  • Validación de importes con tolerancia configurable.

  • Validación de RFC emisor/receptor contra SAP.

  • Soporte multiempresa.

Resultado

Se logró un proceso más controlado y trazable para la recepción documental, reduciendo errores manuales y mejorando la validación previa de CFDI.
