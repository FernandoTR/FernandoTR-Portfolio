Portal de Viáticos

Plataforma web empresarial que digitaliza y automatiza el ciclo completo de viáticos, integrada con SAP Business One para gestionar solicitudes, validaciones fiscales y operaciones contables en tiempo real.

Tecnologías:
ASP.NET MVC C# SQL Server SAP B1 DI API JavaScript / jQuery
Problema

La gestión de viáticos dependía de procesos manuales y acceso directo al ERP, generando errores, baja trazabilidad y dependencia de licencias SAP para tareas operativas.

Solución

Se implementó un portal web desacoplado del ERP que centraliza el flujo de gasto, automatiza validaciones y sincroniza en tiempo real con SAP B1 para la generación de documentos contables y financieros.

Características principales

  • Registro y comprobación de gastos con carga de XML, PDF e imágenes.

  • Flujo de autorización por roles: empleado, autorizador y administrador.

  • Integración con SAP B1 para facturas, pagos, reembolsos y reintegros.

  • Validación fiscal de comprobantes antes del registro contable.

  • Consulta de historial y seguimiento completo de solicitudes.

Mi rol

  • Diseñé y desarrollé la solución fullstack para la gestión de viáticos.

  • Implementé la integración con SAP Business One para sincronización de operaciones financieras.

  • Definí y desarrollé la lógica de negocio para autorizaciones y validaciones del gasto.

  • Participé en despliegue y configuración en entorno productivo sobre IIS.

Retos técnicos

  • Integrar el portal con SAP B1 manteniendo consistencia operativa y contable.

  • Digitalizar reglas de autorización y validación documental.

  • Soportar múltiples empresas y distintos motores de base de datos.

  • Garantizar trazabilidad y control durante todo el ciclo del gasto.

Resultado

La solución permitió reducir procesos manuales, mejorar la trazabilidad del gasto y optimizar la operación al eliminar la necesidad de acceso directo a SAP para usuarios finales.
