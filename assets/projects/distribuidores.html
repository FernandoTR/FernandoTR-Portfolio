Ecommerce Integración SAP Business One Gestión de pedidos

Portal de Distribuidores.

Plataforma web para distribuidores que centraliza pedidos, catálogo de productos y checkout, integrada con SAP Business One para registrar preórdenes de venta de forma automatizada.

Tecnologías:
ASP.NET MVC C# SQL Server JavaScript jQuery HTML/CSS SAP Business One SAP B1 API Crystal Reports PayPal
Problema

El proceso de pedidos necesitaba mayor agilidad, control e integración con SAP Business One.

Solución

Se construyó una plataforma web para generar pedidos de forma autónoma, con catálogo SAP, checkout y registro automático de preórdenes.

Características principales

  • Catálogo de productos sincronizado desde SAP B1.

  • Carrito de compras y flujo de pedido tipo ecommerce.

  • Checkout con pago por PayPal o comprobante bancario.

  • Carga de comprobantes de depósito o transferencia.

  • Registro automático de pedidos como preórdenes en SAP.

Mi rol

  • Desarrollo de módulos frontend y backend.

  • Integración con SAP Business One.

  • Implementación de carrito y checkout.

  • Lógica de negocio para pagos y pedidos.

  • Validaciones y mejoras de experiencia de usuario.

Retos técnicos

  • Integración de artículos y pedidos con SAP B1.

  • Manejo de múltiples métodos de pago.

  • Trazabilidad del flujo de compra.

  • Adaptación del proceso comercial al entorno web.

Resultado

Se optimizó el flujo comercial, reduciendo fricción operativa y mejorando la trazabilidad del proceso de venta.
